# Git Commit Protocol

## CRITICAL RULES
- **NEVER** create a new branch.
- **ALWAYS** commit directly to the `main` branch.
- **ALWAYS** push changes to the remote `main` branch immediately after committing.

## Commit Message Format
Follow conventional commit format:
`<type>(<scope>): <subject>`

Example:
`feat(ui): add ellipsis menu to header`

## Branch Access
- Current default branch: `main`
- Direct push is expected and required.
